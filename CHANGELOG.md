## [1.7.0] - 2026-05-15

### Added
- Add npm audit security tests [[#540](https://github.com/ManageIQ/ui-components/pull/540)]

### Fixed
- Fix drop down default value [[#517](https://github.com/ManageIQ/ui-components/pull/517)]
- Fix deprecated @import usage in SCSS [[#581](https://github.com/ManageIQ/ui-components/pull/581)]
- Fix deprecated division usage (/) in SCSS [[#581](https://github.com/ManageIQ/ui-components/pull/581)]
- Cleanup useless regular-expression character escape [[#580](https://github.com/ManageIQ/ui-components/pull/580)]

### Changed
- Updated from Node 14 to Node 22 [[#525](https://github.com/ManageIQ/ui-components/pull/525), [#569](https://github.com/ManageIQ/ui-components/pull/569)]
- Drop node-sass in favor of sass [[#525](https://github.com/ManageIQ/ui-components/pull/525)]
- Upgrade angular to 1.8.3 [[#532](https://github.com/ManageIQ/ui-components/pull/532)]
- Drop c3 and d3 dependencies [[#539](https://github.com/ManageIQ/ui-components/pull/539)]

### Security
- Bump lodash to v4.18 [[#579](https://github.com/ManageIQ/ui-components/pull/579)]
- Bump lodash from 4.17.21 to 4.17.23 [[#560](https://github.com/ManageIQ/ui-components/pull/560)]
